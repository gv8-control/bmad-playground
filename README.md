# BMad Easy

## Dev Services

Start all services:

```sh
docker compose up -d
```

| Service | URL | Credentials |
|---------|-----|-------------|
| Web app | http://localhost:3000 | — |
| n8n | http://localhost:5678 | `dev@dev.dev` / `dev` |
| PostgreSQL | `localhost:5432` | `postgres` / `postgres` |
